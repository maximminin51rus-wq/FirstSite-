# 🚀 My First Website Project

Welcome to my first web-development project! I'm a 13-year-old student learning how to code on a mobile device. This site is my playground where I practice layouts, styling, and basic interactivity.



## 🛠 Tech Stack
* **HTML5** (Semantics and structure)
* **CSS3** (Flexbox, Grid, Animations, Custom properties)

* **GitHub Pages:** Automatically deployed and live.
